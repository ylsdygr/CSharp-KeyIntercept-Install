CSharp-KeyIntercept-Install

本程序与CSharp-KeyboardIntercept-Zww配套使用，在Windows操作系统中安装键盘拦截程序。

View on GitHub Download .zip Download .tar.gz

欢迎使用键盘拦截安装器

程序功能及使用说明

1.程序可在当前目录中寻找预定义的程序，将其拷贝至指定位置，并在Windows注册中添加自动启动；
2.若程序当前目录中不存在指定预定义程序，则从设定的网络共享地址下载；
3.程序内路径及程序名的修改请参考文件ForInstall.cs中参数定义区域；

程序开发运行环境

1.开发： Windows7 x64 ，VS2012 ，.Net 4.5；
2.运行： .Net 4.5

程序其他说明

注意：
由于操作注册表需要，运行时请以管理员身份运行，否则会导致安装失败；
程序运行需要预定义共享程序服务器IP信息、用户名及密码、预定程序路径、预定程序名；
